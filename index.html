Project: Snapstock

Designed and built a dividend stock analysis platform for Ryne Williams (financial content creator). It's fully a operational platform with real customer adoption ($197/annual subscription).

Speedrun demo:

Features:

  • Company search by ticker
  • Price chart: 5D, 1M, YTD, 1Y, 5Y, 10Y, 30Y
  • Annual and quarterly performance charts (10-year history)
  • REIT-specific charts (e.g., P/AFFO)
  • Financial statements: annual (10-K) and quarterly (10-Q)
  • About company and news

Tech stack:

  • Frontend: Vite, Vue, Tailwind, Shadcn
  • Backend: Node.js, Java (Spring Boot)
  • Databases: PostgreSQL
  • Authentication: JWT (login email)
  • API integrations:
    • Squarespace: customer and order management
    • Financial Modeling Prep: financial data provider (redistribution license)
  • CI/CD: Jenkins
  • Hosting: dedicated server
